### Usage

```js
import Overlay from 'ruik/lib/Overlay';
````

Default Overlay component:

    <View style={{ position: 'relative' }}>
        <Text>Some awesome text</View>
        <Overlay/>
    </View>


Custom Overlay component:

    <View style={{ position: 'relative' }}>
        <Text>Some awesome text</View>
        <Overlay color="#eee" opacity={.35} />
    </View>
